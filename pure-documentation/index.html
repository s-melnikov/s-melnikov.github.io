CSS with a minimal footprint.

Pure.css до смешного миниатюрен. Весь набор модулей умещается в 4.0KB* минифицированного и сжатого в gzip формате кода. Разработаный с учетом мобильных учтройств, нам было важно сохранить небольшой размер исходного файла и каждая строка css была тщательно продумана. Также вы можете экономить еще больше байт используя только подмножество этих модулей.

* Выше мы указали актуальные размеры модулей по отдельности, если их сгрупировать вмести и добавить gzip компрессию они будут занимать еще меньше места.

Ваш базовый CSS.

Pure.css построен на основе Normalize.css и предостовляет стили для нативных HTML элементов плюс стили для самы распространенных компонентов пользовательского интерфейса (UI). Все что вам будет нужно, без ничего лишнего.

С учетом мобильных устройств.

Pure.css является отзывчивым из коробки, поэтому он выглядит замечательно на экранах любого размера.

Не мешает разработке.

Pure.css содержит минимальное колличество стилей и призывает писать стили ваших приложений на его основе.

Создание отзывчивых макетов.

Используя сетки, меню, и другие компоненты, помогает создавать замечательные отзывчивые макеты для экранов всех размеров. Мы создавали их с таким учетом, что бы вам было легко начать ими пользоваться. Просто взгляните на несколько макетов для примера и начинайте ваш следующий web проект на надежной основе.

Создайте свой собственный стиль.

В отличии от других фреймворков, Pure.css очень гибкий, минималистичный и простой. мы считаем что намного проще и удобнее добавить новые правила чем перезаписывать уже существующие правила. Просто добавив несколько строк CSS вы можете изменить внешний вид Pure.css для его использвания в вашем web проекте.
